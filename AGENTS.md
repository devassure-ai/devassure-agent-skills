# Agent Guidance

This repository provides the DevAssure TESTING skills for AI coding agents. Reusable documentation and instructions that help an AI agent install and use the `@devassure/cli` for natural-language, browser-based end-to-end testing.

It follows the [Agent Skills](https://agentskills.io) open standard.

## Repo Layout (high level)

- `skills/devassure/SKILL.md`: the main skill entrypoint (frontmatter + step-by-step instructions)
- `skills/devassure/references/`: supporting reference docs for the skill (CLI reference, troubleshooting, etc.)
- `README.md`: repo overview and installation instructions
- `LICENSE`: MIT license

## Available Skills

- **devassure** — Install, configure, and run end-to-end UI tests with DevAssure CLI.

## Installation
```bash
npx skills add devassure-ai/devassure-agent-skills
```

See `skills/devassure/SKILL.md` for full instructions.
